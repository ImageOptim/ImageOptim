ImageOptim von Kornel Lesiński ist eine GUI für folgende 3rd party utilities:

  • OptiPNG von Cosmin Truta,
  • PNGCrush von Glenn Randers-Pehrson,
  • MozJPEG von Mozilla,
  • Zopfli von Jyrki Alakuijala, Lode Vandevenne,
  • AdvPNG von Andrea Mazzoleni, Filipe Estima,
  • Jpegoptim von Timo Kokkonen,
  • Gifsicle von Eddie Kohler,
  • SVGO by Kir Belevich,
  • and PNGOUT von Ken Silverman.
