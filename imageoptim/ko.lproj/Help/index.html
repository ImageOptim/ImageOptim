어떻게 사용하나요

  1. ImageOptim 화면 안에 PNG / JPEG 파일 또는 폴더를 끌어다 놓으세요.
  2. 완료 될 때까지 기다리세요.
  3. 완료!

ImageOptime의 Dock 아이콘에 파일들을 끌어다 놓을 수 있습니다.

환경설정 변경하는 방법을 배워봅시다..

파일 리스트의 아이콘 설명

이미지가 성공적으로 최적화 되었습니다.
이미 최적화 되어 있어, 더 이상 이미지를 최적화 할 수 없습니다.
파일의 최적화가 진행 중입니다.
다른 파일 최적화 작업이 완료되기를 기다리는 중입니다.
최적화 작업 중에 에러가 발생했습니다. Console.app를 실행시켜 이유를 찾아보세요.

어떻게 동작 하나요.

ImageOptim은 아래 응용 프로그램들을 위한 애플리케이션(GUI)입니다.

  • OptiPNG
  • PNGCrush
  • AdvPNG
  • JpegOptim
  • MozJPEG
  • 그리고 optionally PNGOUT.

ImageOptim은 위의 작업을 수행하고 자동적으로 가장 작은 파일로 만들어줍니다.
